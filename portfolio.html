Visual basic

The City Computer Shop Order Processing Application include a Login Form requiring users to enter correct credentials stored in a text file. A Splash Screen is displayed followed by the Login form. The Order Form features menus for File, Edit, Help, with options to Calculate, Confirm, Clear, and Show Summary. Hardware prices are stored in a 1D array for calculation. The Calculate button/menu calculates subtotals and total order cost. Confirm saves order details to the database, increments order counter, and clears content. Clear resets all fields, and Exit terminates the application. Show Summary displays order information in a Summary Form.
